Reconciliation

This example is using the Telerik Kendo JQuery Grid to display the data from a Json File stored in a share folder named “dta”. On execution, the Console application saves this File inside this folder in order to be accessible for the local web Application. It is using an ASP.Net Web Application just for the Web Server. The page is a simple HTML file it does not require ASP.NET.
Export to Excel: If you wish to export all records to excel use the select "All" items per page from the footer. You can also filter, group and reorder the Columns before Export.
